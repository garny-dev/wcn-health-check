# WCN Organizational Health Check

A modern, gamified version of the Wildlife Conservation Network's Organizational Health Check assessment tool.

## Features

- 🎮 Gamified UI with visual progression stages
- ✨ Smooth animations and hover effects
- 📱 Fully responsive design
- 🌙 Dark mode by default
- 🚀 Built with Vite + React + TypeScript + Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it.

## Development

This is a demo starting with the "Strategic Plan" question. The full form will include:

- Registration & Governance
- Strategic Planning
- Financial Management
- Human Resources
- Program Management
- Communications & Marketing
- Monitoring & Evaluation
- And more...

## Deploy

```bash
npm run build
```

The build output will be in `dist/` - deploy to Vercel, Netlify, or any static host.
